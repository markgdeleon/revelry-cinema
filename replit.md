# Revelry Cinema

## Overview
A film catalog website for Revelry Cinema, showcasing films by Kyle Tims. Inspired directly by A24 Films' clean, catalog-style approach with a nostalgic aesthetic.

## Design Philosophy
- **Catalog-First**: Simple grid of films with titles and years
- **Nostalgic Aesthetic**: Classic, timeless design inspired by A24
- **Helvetica Typography**: Clean, professional sans-serif throughout
- **Films Only**: No photography, no services - purely a film showcase
- **Minimal Interface**: UI stays completely out of the way

## Project Structure

### Frontend (React + TypeScript)
- **Hero**: Minimal site title and "Films" label
- **Film Grid**: Clean grid of film posters with titles and years
- **About**: Brief statement about the filmmaker
- **Contact**: Email and social links only
- **Footer**: Copyright only

### Styling
- **Dark Theme**: Cinematic dark mode as default
- **Tailwind CSS**: Utility-first styling
- **Shadcn UI**: Minimal component usage
- **Typography**: Helvetica Neue system font
- **No Decorative Fonts**: Everything is Helvetica for that catalog feel

## Key Features
1. **Film Catalog Grid**: 2-4 column responsive grid of films
2. **Film Information**: Title and year only - no descriptions
3. **Minimal Navigation**: Just About and Contact
4. **Social Links**: Instagram, Facebook, TikTok, X
5. **Clean Layout**: A24-inspired simplicity

## Recent Changes (October 2025)
- Complete redesign to match A24 film catalog aesthetic
- Removed all photography and service references
- Changed to Helvetica typography throughout
- Simplified to pure film showcase
- Removed filters, categories, and complex interactions
- Made it a simple, nostalgic film catalog

## Tech Stack
- React 18 with TypeScript
- Wouter (client-side routing)
- Tailwind CSS + Shadcn UI
- Express.js backend (minimal)
- Vite (build tool)

## Color Palette
- Background: Deep charcoal (15 8% 8%)
- Card/Surface: Elevated dark (15 8% 12%)
- Primary: Warm orange (25 95% 48%)
- Text: Off-white (0 0% 98%) with muted variants
